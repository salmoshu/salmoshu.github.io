# 二. Issues

# 1. Non-convex MeshCollider with non-kinematic Rigidbody is no longer supported since Unity

# 参考链接：

https://blog.csdn.net/qq_46257993/article/details/106298405

# 解决方法：

  • 方法1：打开刚体中的Is Kinematic；
  • 方法2：关闭Mesh Collider或者将其中的Convex勾选上。

# 2. 将预制体对象作为另一个运动物体的子组件，预制体对象不会随着该物体一起运动

# 问题原因：

预制体勾选了Static，导致无法移动，不勾选即可。
