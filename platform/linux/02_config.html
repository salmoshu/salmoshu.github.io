# 2. 安装拼音

# 参考链接：

https://zhuanlan.zhihu.com/p/399805081

# 基本过程：

首先在manage进行操作，然后注销用户重新登陆，最后在+号处添加拼音智能。

# 3. 修改软件源

Ubuntu 默认的软件源是境外的，速度上会有些问题，我们可以在「Software & Updates」(软件和更新)中选择国内的镜像，如http://mirrors.aliyun.com/ubuntu。

# 4. 管理自动更新

Ubuntu目前会自动下载并安装一些重要的安全更新，一般情况下保持开启即可。如果需要经常自己进行配置更新，而这个自动更新经常导致你“无法锁定管理目录”的错误，则改变自动更新行为。
无法锁定管理目录错误参考链接：https://itsfoss.com/could-not-get-lock-error/

# 5. 控制电脑的自动挂起和屏幕锁定

在开发的时候关闭屏幕锁定会带来更大的便利。

# 6. Ubuntu中安装微信

# 参考链接：

https://blog.csdn.net/m0_54792870/article/details/121489673

# 使用说明：

该链接除了安装说明，还包含了该过程中可能存在的问题，该帖博主将安装与配置过程整理成了shell脚本，使用非常方便。

# env WINEPREFIX="$HOME/.deepinwine/安装后目录名称" /usr/bin/deepin-wine winecfg，比如修改微信的缩放
# 最后调整弹出配置页面中Graphics的Screen resolution值到合适范围即可。
env WINEPREFIX="$HOME/.deepinwine/Deepin-WeChat" /usr/bin/deepin-wine winecfg

# 补充说明：

最后还需要适配高分辨率电脑：
https://blog.csdn.net/qq907482638/article/details/86631738

# 7. Ubuntu启用Nvidia显卡驱动

  • 首先需要在bios页面中禁用secure boot，无法将无法正常使用显卡驱动；
  • 在software update的additional drivers中启用Nidia显卡驱动。
