一. 基本使用 ​

1. HTML Table在线生成器 ​

Markdown在编辑一些比较复杂的表格时候会比较麻烦，一种替代的方案是使用html表格，这里提供一个在线的HTML表格在线生成器。

2. VitePress 启用搜索功能 ​

得益于 minisearch，VitePress 支持使用浏览器内索引进行模糊全文搜索。要启用此功能，只需在 .vitepress/config.ts 文件中将 themeConfig.search.provider 选项设置为 'local' 即可：

ts
import { defineConfig } from 'vitepress'

export default defineConfig({
  themeConfig: {
    search: {
      provider: 'local'
    }
  }
})

3. Vitepress 启用评论功能 ​

3.1 确保已安装必要依赖 ​

确保你已经安装了 vitepress-plugin-comment-with-giscus：

bash
yarn add vitepress-plugin-comment-with-giscus

3.2 获取 Giscus 配置参数 ​

  1. 访问 Giscus 官网。
  2. 登录你的 GitHub 账户，授权 Giscus 访问你的仓库。
  3. 选择你要启用评论的 GitHub 仓库，并启用 Discussions 功能。
  4. 配置 Giscus 参数，例如：
    • 仓库名称（格式：username/repo）。
    • Discussion 分类（例如 Announcements）。
    • 其他选项（如主题、语言等）。
  5. Giscus 会生成一段 JavaScript 代码，其中包含 data-repo、data-repo-id、data-category、data-category-id 等参数，记录这些值。

3.3 修改 index.ts 配置 ​

c
import DefaultTheme from 'vitepress/theme'
import vitepressBackToTop from 'vitepress-plugin-back-to-top'
import { useRoute, useData } from 'vitepress'
import { defineComponent, h, onMounted, watch } from 'vue'
import 'vitepress-plugin-back-to-top/dist/style.css'
import './custom.css'
import './print.css'

export default {
  ...DefaultTheme,
  enhanceApp({ app }) {
    vitepressBackToTop({ threshold: 300 })

    app.component('GiscusTalk', defineComponent({
      name: 'GiscusTalk',
      setup() {
        const route = useRoute()
        const { frontmatter } = useData()
        const getPath = () => route.path || window.location.pathname || '/'
        const getTerm = () => {
          // 优先使用 frontmatter.title，依次回退到 document.title、路径
          return (
            frontmatter.value.title ||
            document.title ||
            getPath()
          )
        }

        const loadGiscus = (term) => {
          const existingScript = document.querySelector('script[src="https://giscus.app/client.js"]')
          if (existingScript) existingScript.remove()
          const existingGiscus = document.querySelector('.giscus')
          if (existingGiscus) existingGiscus.innerHTML = ''

          const script = document.createElement('script')
          script.src = 'https://giscus.app/client.js'
          script.async = true
          script.setAttribute('data-repo', 'username/repo')
          script.setAttribute('data-repo-id', 'your-repo-id')
          script.setAttribute('data-category', 'Announcements')
          script.setAttribute('data-category-id', 'your-category-id')
          script.setAttribute('data-mapping', 'pathname')
          script.setAttribute('data-strict', '0')
          script.setAttribute('data-reactions-enabled', '1')
          script.setAttribute('data-input-position', 'top')
          script.setAttribute('data-theme', 'light')
          script.setAttribute('data-lang', 'zh-CN')
          script.setAttribute('data-loading', 'lazy')
          script.crossOrigin = 'anonymous'
          script.onerror = () => console.warn('Failed to load Giscus script, but comments will be created on submission.')
          document.body.appendChild(script)

          // console.log('Giscus loaded with term:', term)
        }

        onMounted(() => {
          // console.log('Route:', route)
          // console.log('Path:', getPath())
          // console.log('Title:', frontmatter.value.title || document.title || 'No title')
          if (typeof window !== 'undefined') {
            loadGiscus(getTerm())
          }
        })

        watch(() => [route.path, frontmatter.value.title], ([newPath, newTitle]) => {
          console.log('Route changed:', newPath, 'Title:', newTitle || document.title || 'No title')
          if (typeof window !== 'undefined') {
            loadGiscus(newTitle || document.title || newPath || window.location.pathname || '/')
          }
        })

        return () => h('div', { class: 'giscus' })
      }
    }))
  }
}
