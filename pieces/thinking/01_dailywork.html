# 日常工作的思考

# 1. 任务安排

先完成横向任务，再完成纵向任务。

# 2. 中兴项目总结

  1. 比起强调人员的自觉性，更应该在流程上进行改进。
  2. 思考比较少会导致：不自信，不敢想。
  3. 当前所处的氛围不是特别好，需要考虑积累技术，调整一下自己所处的环境，或者参与环 境的改造。

# 3. 日常工作

工作效率低下时的思考，往后工作注意以下几点：

  1. 保持桌面和自身整洁
  2. 设定确定的目标
  3. 保持冷静思考的大脑

# 4. 关于解决问题

对解决问题流程上的几点想法：

  1. 需求出现。
  2. 分析需求，职场新手存在最大的问题就是对需求理解不透彻，所以在有一定理解的前提下，和能力强的前辈进行沟通，确保自己的理解是正确的。
  3. 解决问题，在行动之前，一定要有一个确切的行动方案，并且把它写下来。写得越详细越好，代码可以照着该方案直接写出来。
  4. 落实行动。

最后：比起延长时间，还是要从提高效率上出发。

# 5. 与人沟通方面的思考

  1. 在和人沟通交流的时候，如果没有特殊情况，应当停下手头的事情去和人沟通，这样他人会感觉被尊重。
  2. 做事情不能闭门造车，一来效率很低，二来容易走进死胡同。
  3. 方法论很重要，但是最重要的还是行动力和做事情的激情。

# 5. 关于处理高难度工作

做高难度工作时候的几点方法：

  1. 学习视频或者书籍的时候，一定要把完成时间缩短，不然时间很容易流失，或者看不见成效自己容易灰心丧气。
  2. 在工作当中，不比学校，现在要以目的为导向，提高效率。

# 5. 关于工作激情

激发工作激情的几点方法：

  1. 听劲爆音乐；
  2. 喝茶或咖啡、吃一些类似于蓝莓或坚果的促进物；
  3. 兴趣激励法、正向反馈法。

# 6. 从小事情开始坚持

感觉自己的毅力还是不够，所以想从一些小事情上去坚持，另外由于每天早上都有晨会，自己的表达上缺乏灵活性，所以可以提前做好准备，因此养成每天总结的习惯会很好。

  1. 控制饮食
  2. 每日运动
  3. 每日总结

# 7. 对恒心不足的思考

对于知识掌握的确定性至关重要，不能停留在一个模糊的状态，就像是应付考试临时抱佛脚的时候一样。

类似的，自己身上的很多问题都是由于自己毅力不够，不求甚解所造成的，以后定当言必信，行必果。很多时候自己将原因归咎于心态上，其实做好一件事情，最关键的还是决心和毅力，心态当然能起到很好的促进作用，但是自己不愿意坚持下去，摇摆不定，当然很多事情都做不好。

综上：

  1. 决定做一件事情，非必要情况下，这件事做好了再换下一件事，慎于换方向；
  2. 从小事情开始坚持，如每天锻炼、一本书看完了再换另一本、工作上，目标定了就一定要完成。
